WiinRemote

a program for using Wii Remote on Windows PC.


--how to use
1.connect Wii Remote to PC via bluetooth
2.launch WiinRemote.exe


--see also
http://www.wiili.org/Wiimote


--development environment
Borland Delphi 6 Professional
Windows XP Professional SP2 (Japanese)
DN-BT03D (with BlueSoleil 2.3.0.0)
Nintendo RVL-003


--this program is using following component
HID Controller Suite  ( http://www.soft-gems.net/ )
written by Robert Marquardt
